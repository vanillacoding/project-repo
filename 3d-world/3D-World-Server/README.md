## 📖 프로젝트 사용 방법

1. 디렉토리 이동
   ```
   cd server
   ```

2. package.json에 정의된 패키지 설치
   ```
   npm i
   ```

3. `dotenv` 설정
   - `.env.sample`를 참고하여 `.env` 설정

<br>

4. 실행
   ```
   npm start
   npm run dev // dev
   ```
